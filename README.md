# datascience-casestudy
IMDB Sentiment Analysis
📌 Overview

This project performs sentiment analysis on IMDB movie reviews using data preprocessing, TF-IDF vectorization, and basic visualizations.

🗂️ Dataset
50,000 movie reviews
Labels: Positive / Negative
⚙️ Tech Stack
Python
Pandas
Scikit-learn
Matplotlib
🔄 Process
Load and explore dataset
Clean text (remove HTML, symbols, lowercase)
Convert text using TF-IDF
Create feature: review_length
Visualize data (Histogram, Boxplot, Scatter)
📈 Output
Cleaned dataset
Numerical features for ML
Visual insights on review patterns
🚀 Future Work
Apply ML models
Use deep learning (LSTM/BERT)
Real-time analysis with Kafka & Spark
